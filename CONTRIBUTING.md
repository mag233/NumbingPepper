# Contributing

Thanks for helping improve AI ReadWrite Flow.

## Getting started
- Fork the repo and create a feature branch.
- Keep changes focused and small.

## Branch naming
- `feat/<short-description>`
- `fix/<short-description>`
- `docs/<short-description>`

## Development
```bash
cd ai-readwrite-flow
npm install
npm run dev
```

## Tests and checks
```bash
cd ai-readwrite-flow
npm run lint
npm run test
npm run build
```

## Desktop build (Tauri)
```bash
cd ai-readwrite-flow
npm run tauri:dev
```

## Pull requests
- Describe the problem and solution.
- Include screenshots for UI changes.
- Note any migrations or breaking changes.
- Avoid committing secrets or .env files.
